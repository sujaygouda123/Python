# Python

A collection of small Python learning/practice scripts, organized by date.

## Structure

```
15-07-26/
└── secret_number.py                          # CLI number-guessing game

16-07-26/
├── string_functions_examples.py               # Python string method examples
├── Task 2a - loops, conditionals, functions.py # while/for loops, if/elif/else, functions
└── Task 2b - break, continue, membership.py    # break, continue, in/not in operators
```

## Running a script

Each script is standalone. On Windows, use the `py` launcher:

```
py "16-07-26/string_functions_examples.py"
```

See `CLAUDE.md` for the working conventions this repo follows.
